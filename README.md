# ✅ On-Page SEO Checklist

A complete checklist for optimizing any webpage for search engines.  
Created by **Mehro Nisa** — SEO & Digital Marketing Specialist

---

## 🔍 1. Keyword Research (Before Writing)
- [ ] Identify the main target keyword
- [ ] Find 3–5 related/secondary keywords
- [ ] Check keyword search volume and competition
- [ ] Analyze competitor pages ranking for this keyword
- [ ] Confirm search intent (informational / commercial / transactional)

---

## 📝 2. Title Tag
- [ ] Include the main keyword in the title
- [ ] Keep title between 50–60 characters
- [ ] Make it compelling and click-worthy
- [ ] Avoid keyword stuffing
- [ ] Each page has a unique title tag

---

## 📄 3. Meta Description
- [ ] Include the main keyword naturally
- [ ] Keep it between 150–160 characters
- [ ] Write a clear call-to-action
- [ ] Make it relevant to the page content
- [ ] Each page has a unique meta description

---

## 🔗 4. URL Structure
- [ ] URL is short and descriptive
- [ ] Main keyword is included in the URL
- [ ] No special characters or numbers (unless needed)
- [ ] Words separated by hyphens (not underscores)
- [ ] URL is lowercase

---

## 🏷️ 5. Headings (H1, H2, H3)
- [ ] Page has only ONE H1 tag
- [ ] H1 includes the main keyword
- [ ] H2s and H3s use secondary keywords naturally
- [ ] Headings follow a logical hierarchy
- [ ] Headings are descriptive and helpful to readers

---

## ✍️ 6. Content Optimization
- [ ] Main keyword appears in the first 100 words
- [ ] Content is at least 800 words (for blog posts)
- [ ] Keywords used naturally (no stuffing)
- [ ] Content answers the search intent fully
- [ ] Content is original and not copied
- [ ] Short paragraphs (2–3 sentences each)
- [ ] Bullet points and lists used where appropriate

---

## 🖼️ 7. Images
- [ ] Images have descriptive file names (e.g. `on-page-seo-tips.jpg`)
- [ ] Alt text includes keyword naturally
- [ ] Images are compressed for fast loading
- [ ] No copyright-protected images used

---

## 🔗 8. Internal Linking
- [ ] At least 2–3 internal links added
- [ ] Anchor text is descriptive (not "click here")
- [ ] Links point to relevant related pages
- [ ] No broken internal links

---

## 🌐 9. External Linking
- [ ] At least 1 link to a trusted external source
- [ ] External links open in a new tab
- [ ] No links to low-quality or spammy sites

---

## ⚡ 10. Technical Checks
- [ ] Page loads fast (under 3 seconds)
- [ ] Page is mobile-friendly
- [ ] No duplicate content issues
- [ ] Schema markup added (if applicable)
- [ ] Page is indexed by Google Search Console

---

## 📊 Tools Used
- Google Search Console
- Google Keyword Planner
- Ubersuggest
- Screaming Frog (free version)
- GTmetrix (page speed)
